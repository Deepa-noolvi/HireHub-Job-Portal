# 🚀 HireHub – Job Portal

HireHub is a full-stack Job Portal application that connects **Recruiters** and **Job Seekers** through a secure and user-friendly platform. Recruiters can post and manage job listings, while candidates can browse opportunities, save jobs, and apply by uploading their resumes.

The application features **JWT-based authentication**, **role-based authorization**, and is deployed online using **Vercel** (Frontend) and **Railway** (Backend).

* Update application status

### 👩‍💻 Candidate

* Register and Login securely
* Browse available jobs

* Filter jobs by category and location
* View detailed job information
* Save jobs
* Apply with resume upload
* Track application status

### 🔐 Authentication & Security

* JWT Authentication
* Spring Security

* BCrypt Password Encryption
* Secure REST APIs
* CORS Configuration

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/Deepa-noolvi/HireHub-Job-Portal.git
```

## Backend

```bash
cd backend
mvn spring-boot:run
```

## Frontend

```bash
cd frontend
npm install
npm run dev
```
